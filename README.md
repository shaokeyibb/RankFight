# RankFight
A simple but helpful fight plugin with rank support

## HighLights
1. PlceholderAPI Support
```
%rankfight_rank%
%rankfight_credit%
%rankfight_shopCredit%
```

2. Custom Configuration
```yaml
# 排位积分间隔
CreditInterval: 100
# 胜利获得积分数
CreditWin: 10
# 游戏队列最大等待时间，单位秒，设置为 -1 则无限制
QueueTimeoutSec: 30
```
